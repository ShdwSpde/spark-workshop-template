How It Works

One entry point. Eight specialized agents. Shared state that IS the institution.

YOU │ ▼ ┌─────────┐ │ SPARK │ ← routes to the right agent │ (CAO) │ └────┬─────┘ ┌──────┼──────┐ ▼ ▼ ▼ Agent 1 Agent 2 Agent 3 ... Agent 8 │ │ │ ▼ ▼ ▼ ┌────────────────────────────┐ │ STATE FILES (markdown) │ │ portfolio · priorities │ │ decisions · people │ │ validation · memory │ └────────────────────────────┘

Every agent reads the same files. Every agent writes to them. If the system loses its AI tomorrow, the state files tell the next person everything. Markdown in a git repo — no database, no vendor lock-in, human-readable, diffable.
